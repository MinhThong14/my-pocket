# MyPocket Client

The client-side app of **MyPocket**.

## Getting started

```sh
yarn install
```

### Start with hot reloading

```sh
yarn dev
# or
yarn start
```

### Building for production

```sh
yarn build
```

### Running tests

Tests are TBD.

```sh
yarn test
```
